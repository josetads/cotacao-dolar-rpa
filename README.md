# RPA Monitoramento de Cotação de Moedas

Projeto de automação desenvolvido com Python e BotCity para monitorar a cotação de moedas e registrar os valores em um arquivo Excel.

## Tecnologias

- Python
- BotCity
- Pandas
- Requests
- OpenPyXL

## Moedas Monitoradas

- Dólar
- Euro
- Real

## Estrutura

src/
Database/
requirements.txt

## Execução

python src/main.py
