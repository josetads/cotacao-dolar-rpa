# 💲 Automação de Monitoramento do Dólar (RPA)

## 📌 Descrição
Projeto desenvolvido para automatizar a coleta da cotação do dólar utilizando dados do Banco Central.

## 🚀 Tecnologias
- Python
- BotCity
- Pandas
- Requests

## 📁 Estrutura
- src/: código fonte
- Database/: armazenamento de dados
- resources/: arquivos auxiliares

## ▶️ Execução

```bash
pip install -r requirements.txt
python src/main.py
